# FastAPI Todo App

A simple Todo application built using FastAPI and a modern HTML/CSS/JavaScript frontend.

## Features

* Add a Todo
* View all Todos
* Update Todo status
* Delete a Todo
* FastAPI REST API
* CORS enabled for frontend integration

## Tech Stack

* FastAPI
* Python
* Pydantic
* HTML, CSS, JavaScript

## Run Locally

```bash
pip install fastapi uvicorn
uvicorn main:app --reload
```

Open `index.html` in your browser and start managing your tasks.
